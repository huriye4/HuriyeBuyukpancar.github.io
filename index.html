    Projects

    Category Prediction with Computer Vision

    Accurate product categorization is essential for e-commerce companies to enhance customer satisfaction and optimize sales. To address misclassification issues, a model that predicts product categories from images was developed using advanced deep learning techniques. The project involved extensive data collection, including web scraping from Amazon, Walmart, and eBay to address data imbalance. Datasets of over 18,000 images were created, and transfer learning with pre-trained models like MobileNetV2, VGG19, and ResNet was employed. Data augmentation techniques were applied to improve model performance. MobileNetV2 achieved the highest accuracy, significantly enhancing product categorization and operational efficiency.

    Academic Success Prediction

    In the "Academic Success Prediction" project, a predictive model was meticulously developed using LightGBM to forecast students' academic performance. This initiative entailed comprehensive data preprocessing, sophisticated feature engineering, and meticulous model tuning to address class imbalance and enhance prediction accuracy. Various evaluation metrics were employed to rigorously assess the model's performance, thereby ensuring the provision of reliable insights into factors influencing academic success. This project exemplifies proficiency in machine learning, particularly in addressing data challenges and optimizing models for superior predictive performance.

    Retina Damage Detection from OCT Images

    This project involved the development and comparison of multiple models for retina damage detection from OCT images. It covered model architecture design, transfer learning with InceptionV3, and fine-tuning techniques to improve model accuracy. Technologies and methods used included Python, TensorFlow, Keras, data preprocessing, visual data analysis, custom CNN architecture, InceptionV3 transfer learning, and model tuning and optimization.

    Clustering Analysis Customer Segmentation

    This project aims to analyze customer data from a shopping mall to understand customer behavior and segment them based on various attributes using clustering techniques.
